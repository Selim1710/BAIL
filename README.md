# BAIL
Bangladesh Auto Industries Limited

<h4> Key Features: </h4>
◉ Category and sub-category product and accessories <br/>
◉ user login and registration <br/>
◉ user profile - view order, cancel order, reset password <br/>
◉ add to cart <br/>
◉ admin panel: <br/>

  * Stock management.
  * Category Management.
